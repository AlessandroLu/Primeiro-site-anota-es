Tags de formatação

tag 'h' é ultilizada para títulos e sub titulos, usado somente para diferenciar a importancia do que está escrito, existindo de 1 à 6 (h1,h2,h3,h4,h5 e h6)
tag "p" ultilizada para a criação de paragrafos.
tag "br" ultilizada para 'quebrar linhas' de maneira mais simples.
tag "hr" ultilizada para a criação de uma linha na pagina.
tag "i" ultilizada para deixar a frase em italico-> Ex:Lorem ipsum dolor sit amet.
tag "b" ultilizada para deixar em negrito-> Ex:Lorem ipsum dolor sit amet.
Tag "u" para adicionar uma under line-> Ex:Lorem ipsum dolor sit amet.
tag "small" define um texto menor-> Ex:Lorem ipsum dolor sit amet.
tag "del" adiciona uma linha sob a palavra/frase-> Ex:Lorem ipsum dolor sit amet.
tag "sup" deixar palavra ou numero elevevado-> Ex:Lorem.Lorem, ipsum.
tag "sub" enquanto a sub deixar embaixo-> Ex:Lorem.Lorem, ipsum.
tag "mark" usado para destacar a palavra/frase-> Ex:Lorem ipsum dolor sit amet.
